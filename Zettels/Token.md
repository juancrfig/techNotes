It's a small piece of text that has been converted into a unique ID number so the [[[Large Language Model|LLM]]. 
This is because classifying pieces of text into a set of numbers is cheaper than mapping each possible human world, let alone considering different [[Language|languages]]. Moreover, if *LLM* were only able be trained on complete words, this would make them unable to recognize compose words, slang words, or even get jokes.
